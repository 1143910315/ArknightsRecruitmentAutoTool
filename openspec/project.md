# 明日方舟公开招募助手

## 项目概述
这是一个面向《明日方舟》玩家的桌面软件，用于辅助公开招募决策。软件通过截取游戏界面中的公开招募区域，识别当前可选标签，并根据游戏内公开招募规则推荐最值得选择的标签组合，帮助玩家更高效地筛选目标干员。

## 项目目标
- 通过截图自动识别公开招募标签，减少手动查表成本。
- 根据标签组合推断可能产出的干员范围与稀有度。
- 优先推荐能够稳定获得高稀有度或目标干员的标签组合。
- 以桌面端形式提供简单、快速、低干扰的使用体验。

## 目标用户
- 经常使用公开招募系统的《明日方舟》玩家。
- 不熟悉公开招募标签组合规则的玩家。
- 希望快速判断标签价值、避免误选标签的玩家。

## 核心使用流程
1. 用户在游戏中打开公开招募界面。
2. 用户使用软件截取当前公开招募标签区域，或让软件从屏幕中获取截图。
3. 系统识别截图中的标签文本。
4. 系统基于已知公开招募规则计算可选标签组合。
5. 系统展示推荐结果，包括推荐组合、可能招募到的干员、保底稀有度或特殊价值说明。
6. 用户参考结果，在游戏内手动选择标签并开始招募。

## 核心能力
### 1. 截图输入
- 支持从屏幕截图导入公开招募界面图片。
- 后续可扩展为自动截取指定窗口区域。

### 2. 标签识别
- 识别截图中的公开招募标签文本。
- 需要尽可能适应不同分辨率、缩放比例和部分界面噪声。
- 当识别置信度不足时，应提示用户确认或手动修正。

### 3. 招募规则匹配
- 基于公开招募标签与干员数据进行匹配。
- 支持单标签、双标签、三标签组合分析。
- 能区分高价值标签，例如资深干员、高级资深干员、支援机械等。

### 4. 推荐结果展示
- 输出最优标签组合及其推荐理由。
- 展示每个组合可能出现的干员列表。
- 标注组合的稀有度收益、限定性和风险信息。

## 输入与输出
### 输入
- 用户提供的公开招募界面截图。
- 内置的标签词表、干员数据和公开招募规则数据。

### 输出
- 识别出的标签列表。
- 可选标签组合及对应干员候选。
- 推荐优先级排序与简要说明。

## 设计原则
- 结果优先：核心价值是给出可信、可执行的招募建议。
- 操作简洁：尽量减少复杂配置，让用户截图后即可看到结果。
- 可校验：识别结果和推荐逻辑应尽量透明，便于用户核对。
- 可维护：标签数据、干员数据和规则应独立管理，便于后续更新。

## 非目标
- 不自动操作游戏，也不模拟点击或注入游戏进程。
- 不处理公开招募之外的玩法，例如抽卡、编队、基建等。
- 不依赖联网服务作为唯一识别手段，基础能力应可离线运行。

## 后续建议的规格拆分
- 截图采集与图片预处理
- OCR 标签识别
- 公开招募规则引擎
- 推荐排序与结果展示
- 干员与标签数据维护
